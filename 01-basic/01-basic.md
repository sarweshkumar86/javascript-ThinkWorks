📘 JavaScript Introduction – Questions & Answers

❓ Q1. What is JavaScript?

Answer:
JavaScript is a programming language used to make websites interactive.
It helps in adding features like button clicks, form validation, animations, and dynamic content.

❓ Q2. Why do we use JavaScript?

Answer:
JavaScript is used to make websites dynamic and user-friendly.
Without JavaScript, a website will be static (only HTML content).

❓ Q3. Where does JavaScript run?

Answer:
JavaScript runs inside a web browser like Chrome, Edge, or Firefox.
Each browser has a JavaScript engine (e.g., V8 engine in Chrome).

❓ Q4. What is an Interpreter and a Compiler?

Answer:

👉 Interpreter:

Executes code line by line
Gives output immediately
Shows errors on the same line

👉 Compiler:

Converts the whole code into machine code at once
Then executes the program
Shows errors before execution


❓ Q5. Is JavaScript interpreted or compiled?

Answer:
JavaScript is mainly an interpreted language, but modern browsers also use JIT (Just-In-Time) compilation for better performance.
So, it is a combination of both interpreter and compiler.

❓ Q6. How is code converted into machine language?

Answer:

You write JavaScript code
The browser's JavaScript engine reads the code
It converts the code into machine language (0s and 1s)
Then the computer executes it

👉 Simple line:

“The browser converts our code into machine language so the computer can understand it.”

❓ Q7. What are some features of JavaScript?

Answer:

Form validation
Dynamic content
Fast execution
Platform independent
Easy to learn
